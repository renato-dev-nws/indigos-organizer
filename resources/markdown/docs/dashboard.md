# Dashboard

## Visão Geral

O **Dashboard** é a tela inicial do Indigos Organizer. Ele oferece uma visão geral e resumida das principais atividades do grupo: tarefas, conteúdos e eventos da semana, além de listas com os itens mais urgentes e próximos.

---

## O que aparece no Dashboard

### Programação da semana
Cards organizados por dia da semana (do domingo atual ao próximo domingo) mostrando:
- **Tarefas** agendadas (borda azul índigo).
- **Conteúdos** com publicação prevista (borda roxa).
- **Eventos** da semana (borda verde esmeralda).

Use os filtros de cada seção para exibir apenas o tipo desejado.

### Próximas Tarefas
Tabela com as próximas 10 tarefas atribuídas ao usuário (ou de todos, conforme configuração). Mostra:
- Título com link para a página de detalhes.
- Data agendada (azul) ou deadline (vermelho, se não houver agendamento).
- Ícone de alerta ☠️ com tooltip "Deadline: dd/mm/yyyy" para tarefas próximas do prazo.

### Próximos Conteúdos
Tabela dos próximos conteúdos com data de publicação planejada crescente. Um ícone triste 😢 em vermelho indica conteúdos com data planejada ultrapassada.

### Próximos Eventos
Tabela de eventos futuros com coluna de **presença** indicando se o grupo vai como participante ou audiência. Eventos online exibem um ícone verde.

---

## Dicas de uso

- Use o Dashboard como ponto de partida para cada dia de trabalho.
- Clique nos títulos das tarefas, conteúdos ou eventos para ir diretamente à página correspondente.
