# Calendário Geral

## Visão Geral

O **Calendário Geral** exibe em um único lugar todos os itens agendados do sistema: tarefas, conteúdos, eventos e deadlines. É uma visão integrada para facilitar o planejamento e evitar conflitos de agenda.

---

## O que aparece no calendário

- **Tarefas agendadas** — aparecem na data de agendamento com a cor do módulo de tarefas.
- **Deadlines de tarefas** — aparecem na data de prazo somente se a tarefa estiver vencida ou não tiver data de agendamento.
- **Conteúdos** — aparecem na data prevista de publicação (ou data de publicação real se já publicados), com a cor do status.
- **Eventos** — aparecem na data do evento com a cor do módulo de eventos.

---

## Regras de Negócio

- Tarefas com data de agendamento **não** aparecem como deadline no mesmo dia (evita duplicação).
- Conteúdos publicados aparecem na **data de publicação**, não na data planejada.
- As cores dos itens refletem as configurações de cores dos módulos no sistema.
