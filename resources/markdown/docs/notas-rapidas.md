# Notas Rápidas

## Visão Geral

O módulo de **Notas Rápidas** é um bloco de notas compartilhado por todos os membros do grupo. Use-o para registrar lembretes, listas de tarefas rápidas, ideias provisórias ou qualquer informação que precise estar acessível rapidamente.

---

## Funcionalidades

### Como criar uma nota
1. Clique em **Nova Nota**.
2. Opcionalmente, preencha um **Título**. Se não informado, o sistema gera automaticamente um título no formato "Nota YYYY-MM-DD X".
3. Selecione o **Tipo** da nota (Administrativo, Conteúdos, Tarefas, Planejamento, Outros).
4. Adicione **itens à lista** ou escreva uma **descrição** livre.
5. Marque como **prioritária** se necessário.
6. Clique em **Salvar**.

### Como visualizar as notas
As notas aparecem em cards na página de Notas Rápidas. Cada card exibe:
- O **título** da nota.
- Um ícone com o **número de itens** na lista.
- Um **resumo** da descrição (máximo 40 caracteres, com reticências se ultrapassar).

### Notas arquivadas
Notas que não são mais necessárias podem ser **arquivadas**. Use o filtro para visualizar notas arquivadas.

---

## Regras de Negócio

- As notas são **visíveis por todos os membros** do grupo, não apenas pelo autor.
- Notas arquivadas ficam ocultas por padrão.
